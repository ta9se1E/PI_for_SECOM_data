# 半導体製造データ分析プロジェクト
[English](README.md) | 日本語
## 概要

このリポジトリでは、**製造プロセスにおけるデータ分析の学習と実験の記録**をまとめています。
主に半導体製造プロセスの公開データを対象に、製造データ分析や機械学習の技術習得を目的としています。

本プロジェクトでは以下のスキル習得を目指しています。

* 製造データの分析手法
* センサデータの前処理
* 不良検知（Fault Detection）
* 品質予測（Yield Prediction）
* 製造業における機械学習の応用

このリポジトリは、**学習ログおよび研究ノート**として継続的に更新していく予定です。

---

## 使用データセット

本プロジェクトでは、公開されている半導体製造データセット **SECOM dataset** を使用します。

データ提供元：
UCI Machine Learning Repository
https://archive.ics.uci.edu/dataset/179/secom

データの特徴

* サンプル数：1567
* センサ数：約590
* ラベル：Pass / Fail（製造品質）

このデータは、**半導体製造プロセスにおけるセンサ測定値**をまとめたデータセットです。

※注意
データ本体はこのリポジトリには含めていません。
公式サイトからダウンロードしてください。

## プロジェクトの目的

本プロジェクトでは以下を目的としています。

1. 製造業のセンサデータの特徴を理解する
2. 製造データに対する前処理手法を学ぶ
3. 機械学習による不良検知モデルを構築する
4. 製造プロセスに重要なセンサを特定する
5. プロセスインフォマティクス・製造AIの基礎を習得する

---

## 分析計画

以下のステップで分析を進める予定です。

### 1. データ探索（EDA）

* 欠損値の確認
* センサ分布の可視化
* 相関分析
* 主成分分析（PCA）

### 2. データ前処理

* 欠損値処理
* 定数センサの削除
* スケーリング
* 次元削減

### 3. 機械学習モデル

* Random Forest
* XGBoost
* LightGBM

目的
**センサデータから製品の Pass / Fail を予測する**

### 4. 説明可能AI

* Feature importance
* SHAP解析
* 品質に影響する重要センサの特定
